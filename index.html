1. General Information

This Privacy Policy explains how the Bakımlı Hanım iOS application collects and protects user data. The app only stores data locally on the device using Core Data and does not share personal information with third parties.

2. Contact

Email: info.bakimlihanim@gmail.com

3. Data Collected

The app processes only locally stored data, such as:

  • User-entered care reminders, notes, or product details
  • App settings (e.g., theme, notification preferences)

4. Local Storage (Core Data)

All data is stored locally on the user's device through iOS Core Data. The data is not transmitted to the developer or any external server.

5. Data Sharing

Data is never sold, used for advertising, or shared externally, except when required by law.

6. Analytics & Third Parties

The app contains no external analytics, advertising, or tracking SDKs. If such systems are introduced in the future, explicit user consent will be requested first.

7. Security

Since all information is stored locally, it is protected by iOS system-level security measures. The developer prioritizes user privacy.

8. User Rights

Users can delete or reset their data anytime within the app. For assistance, contact info.bakimlihanim@gmail.com.

9. Changes

This privacy policy may be updated as needed. The most recent version will always be published on this page.
